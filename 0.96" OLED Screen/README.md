### Features:
* OLED self-luminous, no backlight
* Size: 0.96 inch
* Color: Blue
* Voltage: 3.3V-5V DC
* Viewing angle: > 160°
* High resolution: 128 x 64
* Interface Type: IIC / I2C-4PIN

* VCC: OLED Power Positive 3.3-5 V
* SCL: OLED IIC Clock
* SDA: OLED IIC Data Line
